Isto é outro parágrafo

Clique no botão para comparar a posição dos 2 paragrafos.

Possiveis valores de retorno:
1: Os 2 nodes não pertencem ao mesmo document.
2: p1 é posicionado depois de p2.
4: p1 é posicionado antes de p2.
8: p1 é posicionado dentro p2.
16: p2 é posicionado dentro de p1.
32: Os 2 nodes tem nenhuma relação, ou eles são dois atributos no mesmo elemento.

Note: O valor retornado poderia também ser uma combinação de valores. I.e. o valor de retorno 20 significa que p2 está dentro de p1 (16) e p1 está posicionado antes de p2 (4).
