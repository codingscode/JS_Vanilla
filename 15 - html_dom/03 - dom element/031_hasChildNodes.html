Clique no botão para ver se o elemento ul possui alguns descendentes nodes.

Tente remover os descendentes nodes do elemento list, e o resultado será falso invés de true.

Note: Espaço em branco dentro de um node é considerado como text nodes, então se você deixar algum espaço em branco ou linha comentada dentro de um elemento, aquele elemento ainda possui descendentes nodes.
