Clique no botão para acessar o nome node do último descendente node do div.

Um elemento p - 2º descendente no div

Um elemento span - 4º descendente no div

Um elemento p - 1º descendente no div

Um elemento span - 2º descendente no div

Note: Espaço em branco dentro de elementos é considerado texto, e texto é considerado como nodes. Portanto, neste exemplo, o 1º, 3º e 5º descendente do elemento div é um #text node.
